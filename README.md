# turing-tunes
Another ESP32 guitar FX gizmo. Also generates trippy audio-reactive organic visuals (Turing patterns!) by running a reaction-diffusion simulation.

### Parts
ESP32-S3-DevKitC - μcontroller
WM8960 - sound card
ST7735 - SPI LCD module
Limit switch
6.35mm audio jacks x2
5-pin 2D analog joystick

### Schematic
Pin assignments are in main.cpp, lcd.cpp, stick.cpp.
